📚 Documentação Técnica

Documentação completa da plataforma AdTracker MCP — API, Frontend, Deploy e Configurações.

Visão Geral

O AdTracker MCP é uma plataforma de consolidação de dados de anúncios que integra Google Ads, Meta Ads, TikTok Ads e LinkedIn Ads em um único dashboard.

Funcionalidades Principais

  • ✅ Dashboard unificado com KPIs
  • ✅ CRUD completo (Tenants, Clients, AdAccounts, Campaigns, Ads, Metrics)
  • ✅ Sistema de assinaturas com trial de 14 dias
  • ✅ IA integrada via Chat (GPT, Claude, Gemini, Llama)
  • ✅ Exportação CSV, Excel e PDF
  • ✅ Cache Redis e Rate Limiting
  • ✅ Multi-tenant com isolamento de dados
  • ✅ Deploy via Docker e Render.com

Instalação

Backend

cd adtracker-mcp
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python scripts/seed.py
uvicorn src.main:app --reload

Frontend

cd frontend
npm install
npm run dev

Docker

# Iniciar todos os serviços
docker compose up --build -d

# Popular banco de dados
docker compose exec backend python scripts/seed.py

# Parar tudo
docker compose down
